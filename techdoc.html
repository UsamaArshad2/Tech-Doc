Introduction

Today we will be discussing how to use CSS' background color blend mode to bring your webpage backgrounds to a new level. CSS background blend mode allows us to blend a background image and a background color element.

For example:

  • Typing this: main{background-color:blue;} will make the background color blue.


  • Typing this: main{background-image:url("image-url")} will set the background to whatever image you choose.

    • Notice that if you add both to your stylesheet, you will see the background image, but not the background color, because the image will be above the color. What CSS blend mode does, is blend the two.

    To use background blend mode, type this in your CSS:

    main{ background-blend-mode:(value);}

    So what goes into the (value) space? in the following sections I will show you the different values you can place there and the effect they will create on this image.

    Lighten Mode
